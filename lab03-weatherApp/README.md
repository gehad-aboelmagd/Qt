# Weather App ☀️
The application involves a remote request to the [WeatherAPI](https://www.weatherapi.com) & GUI update accordingly.  
It exposes the following features:
1. temperature 🌡️
2. weather condition ⛅
3. wind speed 💨
4. visibility 👁
5. next 5-day forecast ☔
